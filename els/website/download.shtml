Download CVS

Currently (Nov 2001) Easy Linux Server 2.x is still in ALPHA stage. So the most preferred method to get the software is by CVS. You will get the most complete version.

There is an INSTALL file that gives you instructions on how to proceed.

Download RPMs

If you really want almost-finished modules, then you also can download pre-packaged RPMs. Currently (Nov 2001) they don't have achieve a good quality.

What next?

Before you can install the Easy Linux Server modules, you must have a properly installed version of Red Hat 7.2. Therefore please proceed to the Red Hat installation instructions.
