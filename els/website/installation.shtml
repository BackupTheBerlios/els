Installing Red Hat Linux

The Easy Linux Server runs on top of Red Hat 7.2. So you must first install Red Hat. As Red Hat comes with a very good installer and installation manual, we don't really want to repeat that here.

However, we have some tips that aide you in installation, e.g. tips for a good hard disk partition scheme or tips on the selection if modules.
